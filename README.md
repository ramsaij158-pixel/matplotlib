# Matplotlib Practice

A Jupyter Notebook project for practicing basic data visualizations with Matplotlib.

## Project Goal

This repository is focused on learning how to choose and create common chart types for simple datasets.

## Topics Practiced

- Line plots
- Scatter plots
- Bar charts
- Histograms
- Pie charts
- Basic chart labels and titles
- Simple visual interpretation

## Main File

| File | Purpose |
|---|---|
| `Copy_of_matplotlib13.ipynb` | Notebook with Matplotlib chart practice |

## Tech Stack

- Python
- Matplotlib
- NumPy
- Jupyter Notebook

## Learning Outcomes

- Practiced visualizing numeric relationships
- Compared categories using bar charts
- Used histograms to understand distributions
- Created percentage-based pie charts
- Improved chart labeling and presentation basics
